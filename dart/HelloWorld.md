# Hello World


コンソールに "HelloWorld!!" と文字列を表示するプログラムを書いてみましょぅ。



#### 1. DartPadを開く
 https://dartpad.dartlang.org/
 

#### 2. プログラムを書く

```
void main() {
  print('Hello World (1) !!');
  print("Hello World (2) !!");
  print("""
  Hello
  World (3) !!""");
}
```

#### 3. RUNボタンを押す。

```
HTML OUTPUT
CONSOLE
Hello World (1) !!
Hello World (2) !!
  Hello
  World (3) !!
```

と文字列が表示されます。
