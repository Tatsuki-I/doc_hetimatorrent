# TrackerはHttpサーバ
<hr>
<br>

* **TrackerはHttpサーバ**

<br>

* **Get リクエストでデータ**

<br>

<hr>

　これまでの成果で、Torrentファイル から必要な情報を取り
出す事が出来るようになりました。Torrentクライアント は、
Torrentファイルを解析が終わると、次にTrackerにアクセスし
ます。そして、データを配信しているPeerの一覧を取得するの
です。
本章では、実際に簡易のTracker サーバーも実際に作成しなが
ら、Tracker から Peer の一覧を取得する方法について解説し
ます。
